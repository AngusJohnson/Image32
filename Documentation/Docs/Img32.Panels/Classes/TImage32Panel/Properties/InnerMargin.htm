InnerMargin

property InnerMargin: integer; //read only

The amount of space inside a panel's edge that's occupied by bevels and margins.

function TImage32Panel.GetInnerMargin: integer;
begin
  Result := BorderWidth;
  if BevelInner <> bvNone then inc(result, BevelWidth);
  if BevelOuter <> bvNone then inc(result, BevelWidth);
end;
