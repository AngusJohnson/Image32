BkgChBrdColor1

property BkgChBrdColor1: TColor32; //read and write

When BkgChBrdType = pbtChessBoard, the image panel's background will be painted in an alternating chessboard pattern using BkgChBrdColor1 and BkgChBrdColor2. (Default = clLiteBtn32)
