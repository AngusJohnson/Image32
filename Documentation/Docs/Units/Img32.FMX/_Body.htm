Img32.FMX

This unit is needed for file and clipboard access whenever creating multi-device applications using Delphi's FMX framework. This single unit replaces the Img32.Fmt.BMP, Img32.Fmt.JPG, Img32.Fmt.PNG & Img32.Fmt.GIF units that are used in VCL dependant applications.
