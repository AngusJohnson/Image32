TLayeredImage32.GetLayerAt

function GetLayerAt(const pt: TPoint; ignoreDesigners: Boolean = false): TLayer32;

Returns the top-most visible and 'clickable' layer under 'pt', if any.

A layer will be 'clickable' when it inherits from THitTestLayer32 (which most layers do), and when its hit-test mask has been assigned.
