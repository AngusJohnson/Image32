TGroupLayer32


A TRotatingGroupLayer32 is designed to assist rotating layer objects and is returned by calling the CreateRotatingButtonGroup function.

The TRotatingGroupLayer32 contains 2 button layers - a pivot button, and an angle control button. Typically, the position of the angle control button is managed in a MouseMove event, and its new angle (relative to the pivot button) is returned by calling UpdateRotatingButtonGroup.
