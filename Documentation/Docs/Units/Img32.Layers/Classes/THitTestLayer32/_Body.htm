THitTestLayer32

Ancestors

   |
TLayer32

This is an abstract layer. Its descendant classes implement hit-testing such that calls to TLayeredImage32.GetLayerAt in MouseDown and MouseMove events will return the top-most layer under the specified point.

Common descendant layers include - TDesignerLayer32, TRasterLayer32 and TVectorLayer32.
