Index Img32.Layers TButtonDesignerLayer32

TButtonDesignerLayer32.BtnIdx

property BtnIdx: integer; //read only


This is the creation index of the button. This property is useful whenever button order is changed (ie when buttons are moved within the group to alter which button is on top).
