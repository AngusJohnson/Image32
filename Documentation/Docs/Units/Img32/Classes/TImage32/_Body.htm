TImage32

TImage32 is used in just about every graphics operation, either directly though one of its own object methods (and there are many), or indirectly via a function call that takes a TImage32 object as one of its parameters. Key methods include cropping, copying, merge copying, scaling, and rotating, though the full range of methods is listed below.
