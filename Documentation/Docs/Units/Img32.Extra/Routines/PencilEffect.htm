PencilEffect

procedure PencilEffect(img: TImage32; intensity: integer = 0);


Intensity range is 0 - 10

This function converts images into black and white line drawings. 

uses Img32, Img32.Extra, Img32.Fmt.PNG;
...
img := TImage32.Create;
img.LoadFromFile('trees.png');
PencilEffect(img, 1);
img.SaveToFile('trees2.png');
img.Free;
