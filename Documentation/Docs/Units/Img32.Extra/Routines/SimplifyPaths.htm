SimplifyPaths

function SimplifyPaths(const path: TPathD; epsilon: double; isOpenPath: Boolean = false): TPathD; overload;

function SimplifyPaths(const path: TPathsD; epsilon: double; isOpenPath: Boolean = false): TPathD; overload;


This function removes vertices that contribute very little to a path's shape. The epsilon parameter specifies the minimum distance a vertex can be from an imaginary line drawn between its two adjacent vertices.

This function is especially useful following vectorize, that converts monochrome raster images into vector images. This is demonstrated in the VectorizeImage example. It's also very useful following InflatePaths.
