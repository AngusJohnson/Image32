FilterOnColor

procedure FilterOnColor(img: TImage32; color: TColor32);


Removes everything except the specified color from an image, while also preserving where the color is blended with other colors.
This function is the opposite of the RemoveColor function.

Example

uses Img32, Img32.Extra, Img32.Fmt.PNG;
...
img := TImage32.Create;
img.LoadFromFile('beetle_aq.png');
FilterOnColor(img, $FFE45600); 
img.SaveToFile('FilterOnColor.png');
img.Free;
