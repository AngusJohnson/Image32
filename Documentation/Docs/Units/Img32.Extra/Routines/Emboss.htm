Index Img32.Extra

Emboss

procedure Emboss(img: TImage32; radius: Integer = 1; depth: Integer = 10; luminance: Integer = 75; preserveColor: Boolean = false);


Smaller radius is sharper, greater depth increases edge contrast and luminance changes grayscale balance (unless preserveColor = true)

Example 1

  uses Img32, Img32.Extra, Img32.Fmt.PNG;
  ...
  img := TImage32.Create;
  img.LoadFromFile('fruit.png');
  Emboss(img, 1,10);
  img.SaveToFile('fruit_emboss.png');
  img.Free;
